# Zhicong Huang's Academic Homepage

Source for [forwardhuang.github.io](https://forwardhuang.github.io/), deployed with GitHub Pages and Jekyll.

## Content

- About Me and News: `index.md`
- Research Highlights: `_includes/research-highlights.md`
- Selected Papers: `_data/publications.yml`
- Group Alumni: `_includes/alumni.md`
- Gallery: `_includes/gallery.md`
- Site configuration: `_config.yml`

## Local Preview

```powershell
bundle install
bundle exec jekyll serve
```

Then open <http://127.0.0.1:4000/>.

## Theme

Based on the [Minimal Light](https://github.com/yaoyao-liu/minimal-light) Jekyll theme. See `LICENSE` for licensing information.
